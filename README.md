# 🚀 Skilla - Marketplace de Freelancers (Angola)

![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=for-the-badge&logo=mysql&logoColor=white)
![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)
![PHP](https://img.shields.io/badge/PHP-777BB4?style=for-the-badge&logo=php&logoColor=white)

**Skilla** é uma plataforma web freelance desenvolvida especificamente para profissionalizar o mercado de serviços digitais em Angola. O projeto visa resolver a falta de confiança, a desorganização na contratação e a insegurança nos pagamentos entre clientes e freelancers angolanos.

## 🎯 O Problema
Atualmente, o mercado freelance em Angola carece de:
- **Confiança:** Ausência de verificação de profissionais e sistemas de avaliação.
- **Organização:** Comunicação fragmentada via WhatsApp e redes sociais.
- **Segurança Financeira:** Pagamentos informais sem garantias para ambas as partes.

## ✨ Funcionalidades Principais

### 👤 Para Usuários
- **Perfis Distintos:** Fluxos de experiência personalizados para Clientes e Freelancers.
- **Portfólio Digital:** Freelancers podem exibir seus melhores projetos para atrair clientes.
- **Sistema de Busca:** Filtros inteligentes por skills, categorias e avaliações.

### 💼 Para Clientes
- **Gestão de Jobs:** Publicação, edição e encerramento de ofertas de trabalho.
- **Gestão de Propostas:** Análise e seleção de freelancers com base em valor e prazo.
- **Escrow Simulado:** Sistema de retenção de valores para garantir a entrega do serviço.

### 🛠️ Para Freelancers
- **Envio de Propostas:** Sistema de bidding baseado em créditos.
- **Dashboard de Performance:** Acompanhamento de jobs concluídos e média de avaliações.
- **Boost de Perfil:** Possibilidade de destacar o perfil para maior visibilidade.

### 💬 Colaboração e Segurança
- **Chat em Tempo Real:** Comunicação direta com troca de arquivos.
- **Avaliações Bilaterais:** Sistema de feedback (estrelas e comentários) após a conclusão do job.
- **Notificações Internas:** Alertas sobre propostas, mensagens e status de pagamento.

## 🛠️ Stack Tecnológica

- **Backend:** [Laravel](https://laravel.com/) (PHP)
- **Frontend:** [Blade](https://laravel.com/docs/blade) & [Tailwind CSS](https://tailwindcss.com/)
- **Banco de Dados:** MySQL
- **Comunicação:** Laravel Websockets (Mensagens em tempo real)
- **Arquitetura:** MVC (Model-View-Controller)

## 🎨 Identidade Visual
A Skilla utiliza uma paleta de cores focada em confiança e modernidade:
- **Azul Principal (`#2563EB`):** Confiança e Tecnologia.
- **Laranja Accent (`#F97316`):** Energia e Ação.
- **Tipografia:** Inter (Interface) e Space Grotesk (Títulos).

## 🚀 Como Executar o Projeto

*(Seção a ser preenchida após o setup inicial)*
1. Clone o repositório: `git clone https://github.com/seu-usuario/skilla.git`
2. Instale as dependências: `composer install` & `npm install`
3. Configure o `.env`
4. Execute as migrations: `php artisan migrate`
5. Inicie o servidor: `php artisan serve`
